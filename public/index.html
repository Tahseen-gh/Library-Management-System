Session Checklist

  1. Scan patron ID → resolve blockers.
  2. Scan items for checkout → verify due dates.
  3. Collect fines or renew card if flagged.
  4. Finalize receipt and offer holds/renewals.
  5. Process returns → apply fines → route items.
  6. Confirm reshelved status before session end.
